# SuicideDataCopilot
Copilot institucional que evalúa la calidad, uso y acceso a datos en suicidio, mediante el Cuidar Index. Usa un RAG basado en evidencia para hacer las recomendaciones. Interpreta bases de datos considerando criterios éticos, features y potencial de análisis para fortalecer la gestión de datos e investigación local para la prevención del suicidio.
